## Description

Ce plugin permet d'intégrer dans l'entête des pages HTML du blog, des métadonnées sociales, utilisées en particulier sur les réseaux sociaux.

## Réglages

Ce plugin possède une page de réglage, accessible via le menu (item socialMeta dans la section Blog), qui permet d'activer le plugin pour le blog courant et de définir les options souhaitées.

Les métadonnées disponibles sont :

* Open Graph (utilisées par Mastodon, Facebook, …)
* Google (itemprop)
* Twitter/X

Il est possible de définir le compte Twitter et le compte Mastodon à utiliser si fourni(s), la description par défaut si aucune autre n'est trouvée (exemple d'une catégorie sans description) et une image par défaut si aucune n'est disponible en fonction du contexte.

Une option est également disponible pour indiquer que le blog est un photoblog, ce qui permettra d'utiliser la taille originale de l'image utilisée, plutôt qu'une vignette de taille moyenne.
